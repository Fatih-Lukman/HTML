Ini adalah teks span didalam paragraf

Elemen span tidak akan memulai baris baru karena ia memiliki display inline berbeda jika diganti spannya dengan p atau elemen lainnya maka akan melanjutkan elemen lainnya ke bawah

Yang bersifat inline adalah:

  • a
  • img
  • span
  • select
  • label dan input
